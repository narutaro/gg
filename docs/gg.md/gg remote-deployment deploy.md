# gg remote-deployment deploy

Deploy components remotely

## Usage

```bash
gg remote-deployment deploy COMPONENT_VERSION [OPTIONS]
```

## Arguments

#### *COMPONENT_VERSION*

The component version to deploy, updating the deployment.yaml.
